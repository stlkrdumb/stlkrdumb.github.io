# byrai.xyz

Personal blog and portfolio — built with Astro, hosted on GitHub Pages.

---

## Design System

**Editorial + Industrial** aesthetic inspired by independent tech publications and archival documentation.

| Element | Choice |
|---------|--------|
| Display font | **Spectral** (serif) — Google Fonts |
| Body font | **Inter** (sans-serif) — Google Fonts |
| Palette | `#fafaf9` background · `#0a0a0a` foreground · `#d4a017` accent |
| Texture | SVG noise grain overlay (`body::before`) |
| Motion | Staggered CSS keyframe reveals (fade-in-up, letter-spacing spread) |

## Structure

```
src/
├── app/                # Astro App Router (if used)
├── assets/             # Static images & media
│   └── portfolio/      # Hero images for portfolio entries
├── components/         # Shared UI components
│   ├── BaseHead.astro  # SEO, OG, structured data, analytics
│   ├── Footer.astro    # Editorial footer grid
│   ├── Header.astro    # Minimal topbar navigation
│   └── TableOfContents.astro  # Post TOC sidebar
├── content/            # Content Collections API
│   ├── blog/           # Blog posts (Markdown + MDX)
│   └── portfolio/      # Portfolio projects (Markdown)
├── layouts/            # Page templates
│   ├── BlogPost.astro  # Immersive reading room layout
│   └── PortfolioPost.astro  # Editorial post layout
├── pages/              # Route definitions
│   ├── index.astro     # Homepage — cinematic hero + catalog
│   ├── blog/           # Blog listing
│   │   └── [slug]/og-image.png.ts  # Dynamic OG image generation
│   └── portfolio/      # Portfolio gallery
├── styles/             # Global CSS
│   └── global.css      # Tailwind + editorial prose typography
└── utils/              # Helpers
    └── og-image.ts     # OG image rendering logic
```

## Commands

| Command | Action |
|---------|--------|
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build to `./dist/` (includes Pagefind search) |
| `npm run preview` | Preview production build locally |

## Content

### Blog

Posts live in `src/content/blog/`. Each entry requires frontmatter:

```yaml
title: Post Title
description: A one-line summary for listings and OG images.
pubDate: 2025-06-07
heroImage: ./cover.jpg    # Optional — rendered via glob loader + image() schema
readingTime: '~8 min'    # Optional
```

### Portfolio

Projects live in `src/content/portfolio/`. Frontmatter:

```yaml
title: Project Name
description: Brief description of the project.
pubDate: 2025-06-07
heroImage: ./cover.jpg   # Required — used as hero on gallery and post pages
techStack:               # Optional — displayed as pill badges
  - Rust
  - Solidity
```

## SEO & Analytics

- **Canonical URLs** and Open Graph tags via `BaseHead.astro`
- **Dynamic OG images** generated from template (`og-image.ts`) for blog posts
- **Sitemap** auto-generated by `@astrojs/sitemap`
- **RSS feed** at `/rss.xml`
- **Plausible Analytics** — privacy-first, no cookies, GDPR compliant
- **Pagefind** — client-side full-text search
- **Structured data** — JSON-LD (WebSite + Person schemas)

## Hosting

Deployed to **GitHub Pages** with custom domain `byrai.xyz`.

| File | Purpose |
|------|---------|
| `public/CNAME` | Maps GitHub Pages to `byrai.xyz` |
| `public/404.html` | Custom editorial 404 page |
| `public/_redirects` | 301 redirect patterns (reference) |
| `.github/workflows/deploy.yml` | CI/CD — builds and deploys on push to `main` |
